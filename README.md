# Kelsey Day - Author Website

A modern, responsive website for author Kelsey Day to showcase her literary works and connect with readers.

## Features

- **Home Page**: Welcome section with featured works and call-to-action
- **About Page**: Author biography, background, and achievements
- **Works Page**: Comprehensive display of published books and upcoming releases
- **Contact Page**: Interactive contact form and social media links
- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Modern UI**: Clean, professional design with smooth animations

## Tech Stack

- **React 19** - UI framework
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server
- **React Router** - Client-side routing
- **CSS3** - Custom styling with modern features

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build

Build for production:

```bash
npm run build
```

### Lint

Check code quality:

```bash
npm run lint
```

### Preview

Preview the production build:

```bash
npm run preview
```

## Project Structure

```
src/

├── pages/           # Page components (Home, About, Works, Contact)
├── App.tsx          # Main application component with routing
├── main.tsx         # Application entry point

```
